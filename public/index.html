Discord Drive Uploader

Follow the steps in order

  1. 1
    Connect Google Drive

    Authorise the Google account that should own the upload folders.

  2. 2
    Choose a default folder

    Set the fallback location for uploads when a channel uses the default route.

  3. 3
    Add the bot token

    Paste the Discord bot token from the Developer Portal.

  4. 4
    Invite the bot

    Add the bot to your Discord server before choosing channels to sync.

  5. 5
    Link servers and channels

    Map each Discord channel to either a specific Drive folder or the default folder.
