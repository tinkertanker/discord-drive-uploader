Before you start

  • Keep your Discord bot token ready from the Developer Portal.
  • Choose a Google account that should own the upload folders.
  • Invite the bot to your server before mapping channels if you want the server list to load immediately.

Step 1

Connect Google Drive

Start OAuth and allow Drive folder access so the setup can browse the folders you want to use.

Sign in with the Google account that should store uploads

This opens Google's authorisation flow and brings you back here once access is granted.

Step 2

Choose Default Folder

Pick the fallback Google Drive folder used when a channel relies on the default upload location.

Preparing Google Drive picker...

You can either choose a fallback upload folder directly, or tick the option to create a new default-for-discord folder inside the location you pick.

Step 3

Discord Bot Token

Paste the bot token from your Discord application so the uploader can connect to your servers.

Find this in Discord Developer Portal -> Bot -> Token.

Step 4

Link Servers and Channels

Choose the Discord channels this bot should monitor, then decide where each channel should upload in Google Drive.

Default fallback folder

Channels explicitly linked to the default folder will upload here.

If ticked, the folder you choose becomes the parent folder and we will create a new upload folder inside it.

Add a new channel link

Pick a server and channel, then link it to a specific folder or to the default folder above.

You can add as many channel mappings as you want.

Active channel links

Review current mappings, pause syncing temporarily, or remove entries you no longer need.

Step 5

Setup Complete

Your uploader is configured. Invite the bot to Discord and start posting in the channels you linked.

Finalising setup...

Invite the bot

Use this invite to add the bot to your server with the permissions it needs to sync attachments.

After inviting the bot, post photos or videos in a mapped channel and the uploader will move them into the correct Google Drive folder automatically.
